# Resume Projects

This workspace contains three LaTeX resume projects:

- `ai-engineering`
- `software-engineering`
- `forward-deployed-engineering`

Compile all PDFs with:

```sh
make
```

Or compile one:

```sh
make ai
make software
make fde
```

LaTeX is provided by `tectonic`, installed through Homebrew. The BasicTeX cask was attempted first, but its installer requires an interactive sudo password in this environment.
